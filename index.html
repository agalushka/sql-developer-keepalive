SQL Developer 4 keepalive

keepalive extension for Oracle SQL Developer 4+

Project mantained by Stefano Cristalli

Description

This is an extension for Oracle SQL Developer 4, which allows the user to keep a database connection alive by periodically sending a query.

Download

You can download the source code on Github here, or the latest release in ZIP format here.

Setup

Check the wiki for setup instructions (MAKE SURE YOU READ THE WARNINGS).
Currently the instruction only cover the installation from a ZIP archive, a tutorial for recompiling the source code still has to be made (you are welcome to contribute).
